Para poder utilizar el entorno funcional lo primero que deberiamos hacer es teniendo intalado python, iniciar el entorno virtual venv con 
-python -m venv .venv
Luego de esto debemos activarlo, en mi caso que estoy en windows el comando es
-.venv\Scripts\activate
Cuando el venv esta activado en la linea de la consola aparece (.venv) para darnos cuenta
Luego de esto, tenemos que instalar Jupyter Lab con el comando
- pip install jupyter[lab]
Asi ya tendriamos todo lo necesario para trabajar en este entorno.
